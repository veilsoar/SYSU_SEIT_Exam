## INTRODUCTION

鉴于目前找不到一个专门的用于存储SEIT试题以及复习资料的地方，每每需要此类资料之时，都急得焦头烂额，甚而至于把周围人都问一遍的地步。而从高效率的角度来看，上述的操作是毫无疑问的属于重复造轮子一类的，且还会占用自己的有效时间。为了可将更多的时间用于实际学习上而非找复习资料这一重复造轮子的行为上，在此创建了该仓库，用于存储相关复习资料。
如果来者想要把自己手上的资料也分享出来，请随时在**Issues**中创建一个新的Issue，或者直接提交 Pull Request。

非常感谢wj同学提醒，让我意识到需要在这里补充一份说明。由于这个项目里包含了部分大PDF文件，我使用了Git LFS这个工具来管理它们，这可能会给不熟悉LFS的朋友在下载时带来一点麻烦。

如果你 `git clone` 下来的文件打不开，提示“文件损坏”，不要担心，这是缺少了LFS，导致克隆下来的文件将只是无效的指针文件，无法正常打开。

**下面提供两个解决办法：**

- 配置LFS环境。
配置成功后，重新 `git clone` 仓库即可正常获取所有大文件。如果已经克隆，请在仓库目录内执行 `git lfs pull` 来下载文件。
- 直接下载zip包或者Releases版本
